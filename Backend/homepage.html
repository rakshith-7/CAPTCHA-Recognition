Alphanumeric Captcha Recognition Using Convolutional Neural Nets


What Does ACR Do?

Many websites such as email or online shopping sites require users to solve CAPTCHAs, which usually consist of some distorted text that must be read and typed in correctly by the user to distinguish between a human and a machine. For human beings this is might be a comparably simple task, but computers still have difficulties. Completely Automated Public Turing Test to Tell Computers and Humans Apart (CAPTCHA) is an important humanmachine distinction technology implemented in websites to prevent automatic malicious program attacks. It is an established security mechanism to prevent automated posting on internet forums, voting in online polls, downloading files in large amounts and many other abusive usages of web services. It is a network security framework established to gauge any unforeseen threat and prevent such attacks.


The goal of this project is not only to develop a recognition model using Convolutional Neural Nets with minimum hidden layers and still be able to achieve the desired accuracy and efficiency in recognising the alphanumeric characters in the captcha but also to understand what characteristics a captcha should have to be effective.

Input Format

Captcha Image

Captcha Image will be converted to Text Output

User needs to upload an image which consist of a captcha which is in alphanumeric format and the image will be fed to the trained Neural Nets that analyzes the image and convert them into matrices and extract the alphanumeric text given all conditions of blur, strike out, slanting letters and a lot more filters applied across a single letter making it tough to identify them
